## Rodando o projeto

```bash
# Clone esse repositorio
$ git clone https://github.com/Embalo/e-coleta.git

# Entre no repositorio
$ cd e-coleta
$ cd server

# Instale as dependencias
$ npm install

# Desenvolvimento
$ npm run knex:migrate
$ npm run knex:seed
$ npm run dev

```
